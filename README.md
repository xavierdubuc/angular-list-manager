# Installation


Add following line in dependencies of your `bower.json` :

```
"xd-list-manager": "xavierdubuc/angular-list-manager",
```

Add 'xd-list-manager' as a dependency for you app or module that needs it :

```javascript
angular.module('myModule', [
    (...)
    'xd-list-manager',
    (...)
]);
```
